## System interview order

1. Functional and non functional
2. Routes, protected and unprotected
3. UI mockups
4. System diagram
5. Example contract
6. Non functional and performance

## Common non-functional requirements

- Security
- Accessibility
- Observability
- Maintainability
- Performance
  - CDNs
  - Efficient algos
  - Indexing
- Scalability
  - Caching
  - Load balancer
  - Async task queues
  - Autoscaling in cloud
  - Sharding/partitioning
